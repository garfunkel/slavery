Detailed Description

Main libslavery header.

Version
0.1
Authors
Simon Allen
Copyright

License:
MIT

libslavery implements Logitech's HID++ protocol to allow for working with devices such as the MX Master 3 mouse. libslavery provides functions for getting/setting metadata, battery state information, and remapping buttons.
